# ICML 2026 HiLD Workshop Submission Folder

Chosen workshop: 4th Workshop on High-dimensional Learning Dynamics (HiLD), ICML 2026.

Why this is the best fit for the current DriftSSL work: the project is about self-supervised representation geometry, temperature/bandwidth dynamics, Fisher-divergence/score-field interpretation, and high-dimensional learning phases. HiLD explicitly lists feature learning and representation, hyperparameters, statistical mechanics/random matrix theory, high-dimensional stochastic optimization, mean-field/continuous-time limits, and loss landscape geometry.

Official pages checked on 2026-04-15, with the OpenReview submission invitation rechecked on 2026-05-10:

- ICML workshop announcement: https://blog.icml.cc/2026/04/06/announcing-the-icml-2026-workshops-and-affinity-workshops/
- HiLD workshop page: https://sites.google.com/view/hidimlearning/home
- HiLD OpenReview: https://openreview.net/group?id=ICML.cc/2026/Workshop/HiLD
- ICML 2026 author instructions: https://icml.cc/Conferences/2026/AuthorInstructions
- ICML 2026 example paper: https://media.icml.cc/Conferences/ICML2026/Styles/example_paper.pdf

Local structure:

- `main.tex` -- compact workshop skeleton in ICML-style syntax.
- `references.bib` -- local bibliography file.
- `style_files/README.md` -- where to place the HiLD-required style files.
- `figures/README.md` -- where to place plots and diagrams.
- `submission_notes_ru.md` -- short Russian notes about format and submission nuances.

HiLD currently requires its own style file and limits submissions to 5 pages plus supplementary materials. OpenReview currently lists the active deadline as 2026-05-12 15:00 MSK, with PDF uploads capped at 50MB. Use this folder as the working text shell, then place the official HiLD/ICML style file into `style_files/` or next to `main.tex` before compiling.
