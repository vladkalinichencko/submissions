# Official Style Files

HiLD says its own style file is required for both submitted papers and camera-ready papers:

https://sites.google.com/view/hidimlearning/home

The HiLD page links the style file through Google Drive. Download it from the workshop page and place it next to `main.tex` or in this folder.

Do not rely on generic ICML style if the OpenReview form or workshop instructions require the HiLD-specific file.
