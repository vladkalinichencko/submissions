# Official Style Files

Download the official ICOMP 2026 style archive:

https://icomp.cc/storage/settings/FVEGE0aEh6DfhC9Z4igpoBxxfSIAnPSojlBUJK7t.zip

The archive currently contains:

- `icomp2026_conference.sty`
- `icomp2026_conference.bst`
- `icomp2026_conference.tex`
- `icomp2026_conference.bib`
- `math_commands.tex`
- bundled `fancyhdr.sty` and `natbib.sty`

For the simplest local build, put the needed style files in `submissions/icomp-2026/` next to `main.tex`.
